This is the repository for the data analysis, visualization, dashboard, and predictive modelling project based on the Heart failure prediction dataset, by fedesoriano on Kaggle (https://www.kaggle.com/fedesoriano/heart-failure-prediction).

This git contains:
- final_analysis.ipynb, a notebook containing the data analysis, visualization, and predictive modelling
- heart.csv, the source data extracted from Kaggle
- HeartDiseaseTree.png, a visualization of the decision tree resulting from the predictive modelling
- dashboard.ipynb, a notebook containing a jupyter-dash dashboard allowing for interactive data exploration
- dashboard.py, python version of the same
